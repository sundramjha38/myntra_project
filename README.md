# Myntra Clone

A front-end clone of the Myntra e-commerce website built with HTML and CSS.

## Demo
![Demo](assets/myntra-demo.gif)
## Project Structure

```
├── index.html
├── CSS/
│   └── index.css
└── assets/
    ├── myntra-logofinal.webp
    ├── hero_banner.webp
    ├── model_image.webp
    ├── Rising_star/
    │   └── rising_star(1-5).webp
    ├── Grand_reduction/
    │   └── grand(1-6).webp
    ├── medal_brand/
    │   └── brand(1-6).webp
    ├── global_brand/
    │   └── Gbrand(1-6).webp
    └── Category/
        └── cat1-cat33.webp
```

## Features

- Sticky header with logo, navigation, search bar, and action icons
- Hero banner and model image sections
- Rising Stars section
- Luxe Grand Reduction Deals section
- Medal Worthy Brands section
- Grand Global Brands section
- Shop By Category section (33 categories)

## Technologies Used

- HTML5
- CSS3
- Google Fonts — Material Symbols Outlined (icons)

## How to Run

Just open `index.html` in any browser. No build steps or dependencies required.

## Notes

- All images are stored locally under the `assets/` folder.
- The project is static — no JavaScript or backend involved.
